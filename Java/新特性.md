### Java 9

泛型增强  
匿名内部类的菱形括号内不用再写类型

```java

    public interface Generic<T> {
        void test(T t);
    }
    
    public static void main(String[] args) {
        Generic<String> gs = new Generic<>() {
            void test(String s) {
            }
        };
        Generic<?> go = new Generic<>() {
            void test(Object o) {
            }
        }
    }

```

### Java 13

Java Text Blocks

```java

    String sql = "SELECT *"
                   + " FROM user"
                   + " WHERE age = 18";
    ↓
    ↓
    String sql = """
                    SELECT
                    *
                    FROM user
                    WHERE age = 18
                 """;
```
